# NextGen Assets Management System

NextGen Assets Management System is a Laravel + React web application for managing:

- Assets
- Inventory
- Assignments
- Departments
- Categories
- Suppliers
- Users
- Notifications
- Settings
- Profile images

This guide is written for a **fresh clone on Windows** using:

- **VS Code**
- **XAMPP**
- **phpMyAdmin**

---

## 1. What you need before you start

Please install these first:

- XAMPP
- PHP 8.2 or newer
- Composer
- Node.js 18 or newer
- npm
- Git
- VS Code

---

## 2. Clone the project

Open **VS Code**.

Open the terminal in VS Code and run:
